# Ping-Pong-Game
This is a Ping Pong Game application built using html, css, javaScript.

## [Project Link](https://vinay-singhh.github.io/Ping-Pong-Game/)

###### How to play
- Press Enter to start the game
- To control the rods 'A' for left and 'D' for right
